Math Function System

This Project is made to study the accuracy of multiple regression models upon a non-linear mathematical function f(x)= log10(a)+3*b+c5+sin(d) to observe their test results and study the properties of each model and why. This project doesn't contains any GUI since no new data is being added, the prewritten data set is studied in this project. We observe that through this project the RandomForest seems to be most accurate with the accuracy of 99.99999999% while linear regression with only 72% accuracy and Gradient Boosting with 99.99999% accuracy.
